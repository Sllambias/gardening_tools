# gardening_tools
General Tools for Asparagus, Yucca and other plant-based repositories

PyPi:
Tagged releases are uploaded to PyPi: https://pypi.org/project/gardening-tools/
All main commits are uploaded to TestPyPi: https://test.pypi.org/project/gardening-tools/

# Release
The repo will automatically deploy to PyPi for tagged commits on main:
To tag head:
```
git tag 0.x.y
```
and then 
```
git push --tags
```
to push a tag and trigger the deployment workflow.
